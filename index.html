Repair Instructions

Before Sending Your Device:

  1. Backup Your Data - We are not responsible for data loss during repairs
  2. Remove SIM/SD Cards - Remove all personal cards before service
  3. Disable Find My Device - Turn off all device tracking features
  4. Charge Your Device - Ensure at least 50% battery before service
  5. Note Your Lock Screen - Write down your current lock screen code

After Receiving Service:

  1. Factory Reset - Recommended after IMEI/network repairs
  2. Test All Functions - Check calls, data, WiFi, Bluetooth, etc.
  3. Update Software - Install latest OS updates if available
  4. Re-enable Security - Set up new lock screen and security features
  5. Monitor Performance - Report any issues within 7 days

WARNING:

• Unauthorized modifications may violate local laws

• Some features may not work after certain repairs

• Warranty will be void after our service
